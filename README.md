# Install-Python-on-Raspbian
![](https://img.shields.io/badge/language-bash-orange.svg)

### 功能
* 在树莓派官方系统 Raspbian 上安装 Python 3.7.3

* 安装第三方模块
  - [x] scrapy
    - [x] scrapy-deltafetch
  - [x] requests
  - [x] pycryptodome
  - [x] fake-useragent
  - [x] lxml
  - [x] Pillow
  - [x] selenium
  - [x] rsa
  - [x] Crypto

* 删除自带的 Python 2.7 和 3.5

### 运行（root）
```
bash <(curl -L -s https://git.io/fjVM9)
```
